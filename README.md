# Bibliotecas de Livros

Trabalho desenvolvido durante a graduação para a disciplina de **bancos de dados.**

## Tecnologias usadas:
- PHP, XAMPP
- SQL, MySQL
- HTML
- CSS, Flexbox

## Funcionalidades:
- Inclusão, alteração, exclusão e listagem de Bibliotecas;
- Inclusão, alteração, exclusão e listagem de Livros em Bibliotecas específicas.

## Desenvolvimento:
- Diagrama de Classes
- Diagrama de Entidade Relacionamento
- Implementação do código
